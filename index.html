PGObserver

The PGObserver tool was developed at Zalando during the past years to monitor performance metrics of our different PostgreSQL clusters. Due to our use of a stored procedure API layer our strong focus was on monitoring that API layer. However the number of metrics was increased to include other relevant values:

SProc Data

Number of calls.
Run time per procedure.
Self time per procedure.

Table IO Stats

Number of sequential and index scans.
Number of inserts, (hot) updates, deletes.
Table and index size.
Heap hits vs disk hits.

CPU Metrics

CPU load is displayed on the graphs.

Open Source

The PGObserver is one of our python/postgres tools developed and used by Zalando's database team, and we want to share it with you. Go grab it, it's free: to use, to share, to hack.
